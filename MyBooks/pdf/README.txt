DOCUMENTAÇÃO DO PROJETO "ESSE EU JÁ LI!"

Esta pasta contém a documentação do projeto em formato HTML, que pode ser facilmente convertido para PDF.

Arquivos disponíveis:
- user_stories.html: Contém as User Stories do sistema com seus respectivos pontos de história, valor de negócio e testes de aceitação.
- diagramas.html: Contém os diagramas de classes e casos de uso do sistema, com justificativa para a escolha desses diagramas.
- planejamento_iteracoes.html: Contém o planejamento das iterações, com a divisão das User Stories em tarefas, estimativas, tempo real de execução e burndown charts.

Para converter estes arquivos HTML para PDF:
1. Abra cada arquivo HTML em um navegador web (Chrome, Firefox, Edge, etc.)
2. Use a função de impressão do navegador (Ctrl+P)
3. Selecione "Salvar como PDF" como destino da impressão
4. Clique em "Salvar" para gerar o arquivo PDF

Estes documentos fazem parte da entrega do projeto "Esse eu já li!", um sistema de gamificação para leitores desenvolvido como atividade final do curso de princípios de desenvolvimento ágil de software.
